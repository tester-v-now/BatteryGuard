# BatteryGuard

BatteryGuard advanced battery charging control for macOS.

BatteryGuard helps macOS users monitor battery charging and receive notifications when the battery goes above their preferred charge limit.

## Features

- Real-time battery charge monitoring
- Custom battery charge limit notifications
- Lightweight and low resource usage
- Simple and clean macOS interface
- Fast startup and responsive performance
- Designed for long-term battery health awareness
